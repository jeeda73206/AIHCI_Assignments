# AIHCI_Assignments
- In this project, we have used Databricks free edition which is a cloud-based AI service provider allowing us to create traditional chatbot and add advanced capability using artificial intelligence on their platform so that the chatbot can understand more advanced questions that a traditional chatbot cannot answer.
- We have used databricks because it is much easier to use and it is a free edition suitable for students compared to azure which is more complex to configure.
- For the implementation we have used the Python notebook environment inside databricks and then downloaded the final notebook for submission.
